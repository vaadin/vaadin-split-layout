[[vaadin-split-panel.overview]]
= Overview

The [vaadinelement]#vaadin-split-panel# is a Polymer element implementing a split layout for two content elements with a draggable splitter between them.

[source,html]
----
<vaadin-split-panel>
  <div>First panel content</div>
  <div>Second panel content</div>
</vaadin-split-panel>
----
